When deeplabcut installation is done,
Run this line in terminal to launch the GUI: 
    python -m deeplabcut

Create new project
    1. name your project
    2. name of expirementer
    3. specify filepath to where the project will be store
    4. filepath to your videos
        [V] select the videos you want to the project 
    5. [V] copy videos to project folder

Extract frames
    1. Click on extract frames

Train network    
    Change epochs and other parameters
    1. Run deeplabcut.create_training_dataset(config=config_path)
    2. Open DLC Gui Edit pose_cfg.yaml
    3. Change train_settings, epochs

Evaluate
    Creates train_loss
    

Finetune
    1. Go to dlc-models-pytorch and copy your snapshot-(epochs).pt path from your latest iteration 
    2. Paste your path into init_weights line in pose_cfg.yaml file
